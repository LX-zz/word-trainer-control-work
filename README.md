Полнофункциональное приложение для изучения иностранных слов с Express.js бэкендом и React фронтендом.

Функциональность:

✅ Добавление/удаление слов

✅ Тренировка случайных слов

✅ Отслеживание прогресса

✅ Фильтрация по тегам

✅ Статистика обучения
